# 🚀 DevOps Learning Journey (5 Days)

This repository contains my hands-on notes, commands, and practice work from a 5-day DevOps workshop covering Git, CI/CD, Podman, and Kubernetes.

---

## 📅 Day 1 – DevOps Foundations + Linux + Git Basics

* Introduction to DevOps concepts
* Linux basic commands (file system, permissions, navigation)
* Git basics:

  * `git init`
  * `git add`
  * `git commit`
  * `git push`
* Understanding version control

---

## 📅 Day 2 – Advanced Git + GitHub Workflows

* Branching and merging
* Pull Requests (PR)
* GitHub collaboration workflow
* Conflict resolution
* Git best practices

---

## 📅 Day 3 – CI/CD Pipelines with GitHub Actions

* Introduction to CI/CD
* Creating workflows using GitHub Actions
* Automating build and test
* YAML basics for pipelines

---

## 📅 Day 4 – Podman Deep Dive

* Introduction to containers
* Podman vs Docker
* Basic commands:

  * `podman build`
  * `podman run`
  * `podman ps`
* Running applications inside containers
* Containerizing a Node.js app

---

## 📅 Day 5 – Basic Kubernetes

* Introduction to Kubernetes (K8s)
* Creating clusters using KIND
* Basic commands:

  * `kubectl get pods`
  * `kubectl apply`
* Deploying applications on Kubernetes
